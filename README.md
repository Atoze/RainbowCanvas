# RainbowCanvas

ブラシの色をキャンバスの座標に合わせて虹色に描画してくれる、ブラシスクリプトです。<br>
メディバンペイント、FireAlpacaなどで使えます。
 
## How to Use
 
RainbowCanvas.bs をブラシ一覧に ドラッグ & ドロップ で使えるようになります。<br>
RanbowCanvas_Simple.bs は機能を制限した簡易版になります。
